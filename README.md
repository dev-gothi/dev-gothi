## What I'm working on

Currently focused on digital design using Verilog and SystemVerilog — building and properly simulating FSM-based designs, and documenting each project on GitHub.

## Projects

**[Verilog-Projects](https://github.com/dev-gothi/Verilog-Projects)**
Collection of digital design projects in Verilog/SystemVerilog, simulated and verified. Includes:
- UART TX/RX system with 16x oversampling
- Synchronous FIFO
- Vending machine FSM (parameterized pricing)
- Mealy sequence detector (overlapping pattern detection)
- Debouncer / pulse filter FSM
- Digital door lock FSM with password update
- 4-floor elevator controller (Moore FSM)
- Shift registers (SIPO, SISO, PISO, PIPO, Universal)

**Python Projects**
CLI-based applications built while learning object-oriented programming:
- Library Management System
- Bank Account System
- Contact Book
- ATM System

**C Projects**
Data structures and algorithm practice, including custom implementations of Stacks and Queues, along with problem-solving on LeetCode (arrays, linked lists, strings).
